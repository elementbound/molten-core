# Molten Core #

A WebGL demo powered by [three.js](https://threejs.org/) and the magic of shaders.

<p align="center">
    <img src="assets/small.gif" />
</p>

## Usage ##

* Install dependencies with `npm install; npm install --only=dev`
* Run with `npm start`
